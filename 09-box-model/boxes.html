Learn about the Box Model

Description

The Box Model is a sample box that shows how every element wraped. It ha a border, a padding, a margin, and a content. It has two types of boxes:an inline and block. It is all about styling an element and make everything measured properly. As the MDN website says:"Everything in CSS has a box around it, and understanding these boxes is key to being able to create layouts with CSS, or to align items with other items."
