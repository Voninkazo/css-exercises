Transition

  • The transition property is a 'shorthand property'. What other properties are included in this shorthand property?
    • transition-property
    • transition-duration
    • transition-timing-function
    • transition-delay
  • What is the difference between a timing function, delay, and duration?
    • duration:sets exactly how long the animation should take.
    • timing-function:duration between each element start acting.
    • delay:duration to wait before the animation starts.
  • What CSS properties are animatable using transitions? These shorthand properties:
    • backgruond-color
    • color
    • margin
    • border
    • gap
    • fonts
    • width
    • padding
    • scale
    • scroll
    • text
    • transform
    • vertical-align
    • word-spacing
    • z-index
    • zoom
    • top
  • Can we select more than one property to transition at a time? If so, how?
    • Yes, we can select more than one property by using commas between them as the values of the "transition-property".
    • We can also use the "all" keyword to iclude all properties.
  • How can we set or define the desired transition state, using CSS?
    • Using pseudo-classes like :
      • :hover
      • :active
      • :target
      • :focus
      • :checked
    • We also can use id selectors.
  • Do we need transitions? What makes them useful?
    • It help us to control the animation speed when we change CSS properties.
    • It also lets us to choose what CSS properties to animate ,when the animation starts by using the transition-delay property.
  • At what point can transitions become a problem?
    When we have to wait.
  • Transitions don't work reliably with the auto keyword being one of the end states. Why is that?
    • Because transition only works with numerical values.
    • It is the browser which has to make the calculation.
  • How can we stagger transitions? i.e. make a number of child elements all start at different times?
    We can set a transition-delay property on each element. Then we increase the start time according to what we want to go first.
