Overflow

Look at these properties

Overflow

The default value is visible.It is useful when we have a lot of writings that cannot fit in the context.

Clip

The default value is `auto`. It defines tha visible portion of an element. It only works with an absolute position elements.

Clip-path

It creates a clipping region that sets what part of an element should be shown. Parts that are inside the region are shown, while those outside are hidden. The default value is `auto`. These are the the values:url(), circle, ellips, inset,polygon, path.

Text-overflow: ellipsis

This property sets how hidden overflow content is signaled to users. It can be clipped, display an ellipsis ('…'), or display a custom string. "The keyword value `ellips` will display an ellipsis ('…', U+2026 HORIZONTAL ELLIPSIS) to represent clipped text. The ellipsis is displayed inside the content area, decreasing the amount of text displayed. If there is not enough space to display the ellipsis, it is clipped." This only works with `white-space` property.

General questions

What is the browser's default way to handle contents that are larger than its container?

The browser uses the auto value of the overflow property.
